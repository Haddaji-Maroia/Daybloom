## 📱 Présentation de l'application

Daybloom est une application de journal personnel qui vous permet de capturer
vos pensées, vos émotions et vos souvenirs du quotidien. Écrivez chaque jour,
construisez une série et débloquez des badges au fil de votre progression.

---

## 👤 Public cible

Les jeunes qui souhaitent un espace simple et personnel pour écrire sur leur
journée, suivre leurs émotions et garder leurs souvenirs au même endroit.

## 🛠️ Installation

### Prérequis
- Flutter SDK >= 3.5.0
- Dart SDK >= 3.5.0
- Xcode (pour le simulateur iOS)

### Étapes

```bash
git clone https://github.com/Haddaji-Maroia/journal-app.git
cd journal-app/Code/daybloom
flutter pub get
flutter run
```
